Arduino Macro Pad Editor

Current version: 0.7.7

How to start - Windows

To use AMP-E, simply run the AMP-E.bat

How to start - Mac/Linux

To use AMP-E, simply double click on the jar file

How to use AMP-E

Once the program is started, you will be presented with the COM select screen. Search for the one named AMP v1. This will then load up the editor. Here you are presented with 20 buttons. When selecting one, you get the option to change the MIDI note and change the RGB LED (if you have them on your product). In order to confirm your entry, press Enter. To push it to your AMP, press Push To COM.

To save your configuration, hit Save Config, this will create a config.ampe file, which contains your configuration, which you can later re-open using the Load Config button.

Make sure that no other devices or programs are using the AMP. If so, AMP-E will not start
